Minecraft Bot for Because

一个功能丰富的Minecraft机器人，集成了Kook聊天平台与智谱AI对话功能，提供游戏内外的互动体验。

开始使用 了解更多

功能特性

游戏机器人功能

  • 自动化操作
  • 玩家监控
  • 事件响应
  • 智能聊天
  • 定时任务

Kook集成

  • 双向消息同步
  • 实时通知推送
  • 多频道管理

Web实时界面

  • 用户管理与认证
  • 消息管理
  • 日志查看
  • 实时对话

AI聊天功能

  • 智谱AI集成
  • 多用户会话
  • 上下文记忆
  • 角色扮演

安装与配置

环境要求

  • Python 3.8+
  • SQLite数据库
  • Kook机器人Token（可选）
  • 智谱AI API密钥（可选）

1. 克隆项目

git clone <repository-url>
cd mc_bot_for_github

2. 安装依赖

pip install -r requirements.txt

3. 配置环境变量

将.env.example文件改为.env文件并配置必要参数

# 游戏连接配置
PLAYER: Minecraft玩家名
HOST: 服务器地址
PORT: 服务器端口

# Kook配置
KOOK: Kook机器人Token

# 智谱AI配置
ZHIPU_AI_API_KEY: 智谱AI密钥

使用方法

启动机器人

启动Minecraft机器人

python mc.py

启动Web服务器

python server.py

访问Web界面

http://localhost:211

常用命令

  • /tpa (bot_name) 承接传送
  • /whisper (bot_name) 与AI进行对话
  • 关键词触发 发送特定关键词触发彩蛋功能
